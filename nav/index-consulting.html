Visão geral da API da NASA

API são interfaces de programação de aplicativos. Elas nos ajudam a trocar informações entre máquinas utilizando a web As APIs da NASA permitem que os desenvolvedores usem os dados da NASA, como imagens, para o desenvolvimento de aplicativos. Para isso é nessesário solicitar uma chave de acesso no site da agência. A partir dessa chave você poderá fazer a requisição ao servidor da corporação para receber as fotos, explicações, vídeos que estão armazenados.
